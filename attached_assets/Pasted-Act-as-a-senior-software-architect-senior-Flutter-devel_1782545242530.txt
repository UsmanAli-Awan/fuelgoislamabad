====================================
PROJECT NAME
====================================

FuelGo Islamabad

====================================
TARGET
====================================

Islamabad only.

Only registered petrol pumps can sell fuel.

Customers can order fuel from nearby petrol pumps.

Every petrol pump has its own delivery riders.

There is NO separate rider application.

The petrol pump manages its own riders internally.

When delivery is completed, the petrol pump owner manually changes the order status to Delivered.

No OTP confirmation is required.
